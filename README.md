# 3D形状生成WEBアプリ

## 概要
2行の表データから任意のルールに従って点群を計算し、その点群とy=0平面で囲まれる3D形状を可視化するWEBアプリケーション。

## ファイル構成

```
aminui_maker/
├── app.html                    # メインHTMLファイル
├── app.css                     # メインスタイルシート
├── app.js                      # メインアプリケーションロジック
├── modules/
│   ├── tableManager.js         # 表管理モジュール
│   ├── pointCalculator.js      # 点群計算モジュール
│   └── shape3DRenderer.js      # 3D描画モジュール
└── README.md                   # このファイル
```

## 基幹機能

### 1. 表入力機能 (TableManager)
- 画面上半分に配置
- 2行の表でデータ入力
- 列の追加・削除
- 値の増減ボタン
- リアルタイムバリデーション

### 2. 点群計算機能 (PointCalculator)
- 表データから点群を生成
- 複数の計算ルールをサポート:
  - 円筒配置
  - スパイラル配置
  - 波形配置
  - カスタムルール
- 点群の補間と検証

### 3. 3D形状表示機能 (Shape3DRenderer)
- 画面下半分に配置
- WebGLによる3D描画
- 点群とy=0平面で囲まれる立体を生成
- インタラクティブな視点操作:
  - マウスドラッグで回転
  - ホイールでズーム
  - 右ドラッグでパン
- 表示オプション:
  - グリッド表示
  - 座標軸表示
  - ワイヤーフレームモード
  - ライティング調整

## データフロー

```
表入力 → 点群計算 → 3D形状生成 → 描画
  ↓         ↓           ↓          ↓
2行表    点群配列   メッシュデータ  画面表示
```

- HTML5

- JavaScript (ES6+)
- WebGL (3D描画)
